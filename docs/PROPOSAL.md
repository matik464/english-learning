# EnglishWords - Project Proposal

## Vision

EnglishWords is an open-source English vocabulary learning platform built around the Oxford 3000 and Oxford 5000 word lists.

The project combines language learning with modern software engineering practices and serves both as a practical learning tool and a portfolio project.

---

## Problem Statement

Many learners struggle with:

- remembering vocabulary consistently,
- tracking learning progress,
- finding high-quality vocabulary lists,
- maintaining long-term motivation.

Oxford 3000 and Oxford 5000 provide excellent vocabulary foundations, but they are distributed as static resources and do not offer interactive learning experiences.

EnglishWords aims to transform these vocabulary lists into a modern learning application.

---

## Target Audience

### Primary Audience

- English learners (A1–C1)
- Self-taught learners
- Students

### Secondary Audience

- Developers interested in language learning
- Recruiters and hiring managers evaluating the project

---

## Objectives

### Learning Objectives

- Learn vocabulary through repetition.
- Build long-term retention.
- Track learning progress.
- Provide meaningful statistics.

### Technical Objectives

- Demonstrate clean architecture.
- Demonstrate TypeScript expertise.
- Demonstrate data processing pipelines.
- Demonstrate API integrations.
- Demonstrate state management patterns.

---

## MVP Scope

The first version should include:

- Oxford vocabulary import
- Vocabulary browsing
- Flashcards
- Progress tracking
- Local persistence
- Statistics dashboard
- Translation support

---

## Out of Scope (Initial Release)

The following features are intentionally postponed:

- User authentication
- Cloud synchronization
- Mobile applications
- AI-powered tutoring
- Social features

---

## Success Metrics

The project will be considered successful when:

- vocabulary data is fully imported,
- users can learn through flashcards,
- progress is persisted,
- statistics are available,
- the architecture remains maintainable,
- the repository showcases production-quality code.
